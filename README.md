# rdx_weaponshop

## Requirements
- [warmenu](https://github.com/RalivTV/warmenu)

## Download & Installation

- Download https://github.com/Redm-Extended-PT/rdx_weaponshop
- Put it in the `[rdx]` directory

## Installation
- Add this to your `server.cfg`:

```
ensure rdx_weaponshop
```
